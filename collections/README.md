# API Collections

## Bruno (Recommended)

[Bruno](https://www.usebruno.com/) is a free, open-source API client. To use the Schelling collection:

1. Install Bruno: `brew install bruno` or download from usebruno.com
2. Open Bruno → "Open Collection" → select `collections/schelling-api/`
3. Select the "production" environment
4. Start with "Describe Network" to explore

The collection covers the core workflow: describe → seek/offer → interest → commit → contract → deliver → report.

## Postman / Insomnia

Import the OpenAPI spec directly:
- **URL:** https://www.schellingprotocol.com/openapi.yaml
- Postman: Import → Link → paste URL
- Insomnia: Import → From URL → paste URL
